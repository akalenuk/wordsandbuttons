Programmer's guide to polynomials and splines

So you are a programmer. Why would you want to know about polynomials? Mathematical analysis explained with Python, blood, and TNT shows how you can analyze and synthesize arbitrary functions as polynomials. But it doesn't have to be ready made functions. Sometimes you would want to model an arbitrary function out of some points or some properties like tangents of curvatures. Like when you want to do some animation or nice video effect or simply draw a curve that goes through the points you want and looks just as you want it to.

Polynomials or even polynomial splines might not always be the best tool for the job but they posess some traits that programmers value. They are inherently simple, versatile, and, most noticably, highly effective in terms of performance. Let's say we have a polynomial like that:

P(x) = ax3 + bx2 + cx + d

It only takes 6 multiplications and 3 additions to calculate, and this is important because your model is meant to be calculated a lot. But even this can be reduced by using Horner's method. The same polynomial may be written as:

P(x) = ((ax+ b)x + c)x + d

And that's only 3 multiplications and 3 additions. See, we barely started, and you already learned how to throw away one third of computation effort.

Polynomial interpolation

Fitting a n-degree polynomial in exactly n+1 points is called polynomial interpolation. There are several ways to do that. You can use Newton polynomials or Lagrange polynomials for that. But the very basic way you can get an interpolating polynom is by solving a linear system that it automagically makes itself.

If a polynom P(x) goes through the point (xi, yi) then, obviuosly enough, we can claim that P(xi) = yi. Say we want to fit a polynomial into a set of three points. This means that:

P(x1) = y1
P(x2) = y2
P(x3) = y3

Generally, we can not fit a staight line into three arbitrary points. We would have to bend it forming a parabola. Or, put in another words, 2-nd degree poynomial also known as quadric.

P(x1) = ax12 + bx1 + c = y1
P(x2) = ax22 + bx2 + c = y2
P(x3) = ax32 + bx3 + c = y3

Since we know xs and ys, we only have to solve the system for (a, b, c) coefficients and since it's a three equation system with three variables, it should generally give us one and only one solution.

Here, try it yourself. Here is the canvas you can drag the points on however you like:

P(x) = ax2 + bx + c

This is also a good mental model for understanding linear systems. Generally, you can't fit a straight line into three points just like you can't find a solution for (n-1) variables for n-equations system. But sometimes you can. It's when some of the point coincide of they all deliberately lie on a straight line.

And the vise versa it's even more exiting. We can fit an infinite number of parabolas into a set of only two points. We just can't choose one of them, that's why can't unambiguously solve an n-equations system for (n+1) variables.

But what if we can? What if we introduce some other criteria to choose the best fit?

Sythesis

And this brings us to a field of polynomial synthesis. It is something between poynomial series and polynomial interpolation. With polynomial series we can model a function based on its derivatives at some point, and with synthesis we can use both points and derivatives. And even more than that actually, but that's a whole different story.

Function's derivative is closely related to a geometric properties of its plot. First derivative sets a tangent, and the second one sets a curvature.

Let than say we want a function that goes throuth a pair of points and has a specific tangent in both points. We can easily synthesize this function as a polynomial.

Just like before, we would have to set a system of equations. Now we want four conditions, so we should pick a 3-rd degree poynomial — a cubic.

P(x1)' = 3ax12 + 2bx1 + c = dy1/dx
P(x1) = ax13 + bx12 + cx1 + d = y1
P(x2) = ax23 + bx22 + cx2 + d = y2
P(x2)' = 3ax22 + 2bx2 + c = dy2/dx

Some of the equations are formed from the points and some from derivatives. You could also add integrals if you wanted to constaint some integral properties just like that so this is a rather powerful technique.

But anyway this gives us some way to form a smooth continuous function between two points with tangental constraints at these points. For now, let's stick to that.
