Trippy polynomials in arctangent scale

This is a cubic parabola.

x3 + x2 - 4x + 2

You can see that it intersects the x-axis at least two times, and it has at least one local minimum. However, this is not the whole truth. It intersects the x-axis exactly 3 times, it has exactly one minimum and one maximum, it's just the plot fails to show that. You can drag-scroll it to the left or zoom out with shift + mouse wheel to find the missing intersection. The plot is interactive but it can only show a small piece of ℝ2 space.

Now let's change the plot scale. Instead of a linear scale, we'll use the arctangent scale which shows all the function from -∞ to +∞. Now the same function looks like this.

Try dragging, double click, and shift+mouse wheel if you're browsing from a PC.

Yes, it looks a little unconventional but you can clearly see that it's domain and range are both the whole ℝ. You can see that is has tree roots, one local minimum and one local maximum. Arctangent scale leaves no mystery behind, it shows all there is about this function.

So let's look at the polynomials using arctangent scale plots.

X-axis intersections and roots

When a function plot intersets the x-axis, this is the same as.

ax3 + bx2 + bx + d = 0

Another words, each intersection is a root of the equation with the right side being 0.
